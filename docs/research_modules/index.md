# Research Modules

Research modules extend PyPSA-GUI with custom analysis, modelling, or visualisation tools.

They allow users to:

- Add new scientific methods
- Prototype ideas quickly
- Share reusable research workflows

## Examples

- ML surrogate
- Reserve market modelling
- Scenario generation
- Flow tracing
- Battery degradation analysis


## Philosophy

- Keep core GUI simple
- Enable advanced features via modules
- Encourage open research contributions
