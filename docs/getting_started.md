# Getting Started

## 1. Launch the GUI
pypsa-gui

## 2. Load a network
- File → Open → select .nc file

## 3. Explore components
- Go to "Components"
- Inspect buses, generators, etc.

## 4. Run optimisation
- Go to "Run → Optimisation"
- Click "Run"

## 5. View results
- Go to "Analysis → Summary"
- Explore plots and KPIss
